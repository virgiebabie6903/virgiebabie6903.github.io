POW Block (WIP)

This object entails pretty much everything I set out to do with this project. I haven't worked much with visual effects up to this point, but have always had an interest in learning, so I figured that something with an obvious place to add some visual flair would be an auto-include with this project.

Largely inspired by Mario Kart World's addition of "BOOOM" text over Bob-Omb and Blue Shell explosions.
